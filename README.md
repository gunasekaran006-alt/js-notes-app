# 📝 Modern Task Manager (CRUD App)

A sleek, responsive, and fully functional Todo application built using vanilla web technologies. This project implements full **CRUD** (Create, Read, Update, Delete) operations, helping you keep track of your daily tasks with a clean modern user interface and a built-in dark mode.

## ✨ Features

- **Create:** Easily add new tasks to your list.
- **Read:** View your active and completed tasks. All data is saved in the browser's `LocalStorage`, so you never lose your data even after refreshing the page.
- **Update:** Click the ✏️ icon to edit existing task text, or mark tasks as 'Done' with the custom checkbox.
- **Delete:** Click the 🗑️ icon to remove specific tasks, or use the 'Clear Done' button to remove all completed tasks at once.
- **Dark/Light Theme:** Seamlessly toggle between beautiful dark (Slate) and light (Teal) themes.
- **Task Filtering:** Filter your view by 'All', 'Active', or 'Done' tasks.
- **Dynamic Counter:** Keeps track of how many pending tasks are left to be completed.

## 🛠️ Technologies Used

- **HTML5:** Semantic page structure.
- **CSS3:** Custom UI design, CSS variables, smooth transitions, and responsive layout.
- **Vanilla JavaScript (ES6+):** DOM manipulation, event handling, and LocalStorage logic (No external libraries used).
